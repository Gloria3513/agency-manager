# 🎓 스마택트 교육 프로그램 견적 시스템

고객이 직접 교육 프로그램을 선택하고 맞춤형 견적을 확인할 수 있는 공개 페이지입니다.

## 📱 배포 주소

- **Streamlit Cloud**: (배포 후 입력 예정)

## 🛠️ 로컬 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🎯 주요 기능

### 교육 프로그램
- 🤖 **AI 자율주동차 (큐트봇)**: AI 자율주행차 코딩
- ⌚ **마이크로비트 메이커**: 시계 및 기 만들기
- 💻 **엔트리/스크래치 코딩**: 블록 코딩 입문
- 📱 **디지털 리터러시**: 공공 디지털 기초
- 👨‍👩‍👧 **학부모 특강**: 우리 아이 AI·코딩 교육법
- 👨‍🏫 **교사 연수**: 교육 현장 활용 SW·AI

### 자동 견적 계산
- **강사료**: Type A(15만), B(8만), C(4만), D(3만원/시간)
- **장비비**: 마이크로비트 장비셋 10만원
- **교구비**: 큐트봇(1.5만원), 시계(1만원), 워크북(5천원/인)
- **20명 초과 시 보조강사 자동 추가**

## 📊 견적 계산 예시

### 초등학생 20명, 큐트봇 수업 2차시
- 강사료: 8만원 × 2 = 160,000원
- 장비비: 마이크로비트 셋 = 100,000원
- 키트비: 큐트봇 1.5만원 × 20명 = 300,000원
- 워크북: 5천원 × 20명 = 100,000원
- **총계: 660,000원**

### 학부모 30명, 특강 2시간
- 강사료: 15만원 × 2 = 300,000원
- 워크북: 5천원 × 30명 = 150,000원
- **총계: 450,000원**
